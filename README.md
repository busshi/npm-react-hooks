# React-hooks

Here is some reusable hooks for ReactJS and NextJS

- useGetElementDimensions
- useGetScrollDimensions
- useIsChrome
- useIsDarkMode
- useIsMobile
- useIsScrolling
- useIsUserInactive
- useListenForOutsideClick
- useWindowSize

## Installation

`yarn add @busshi/react-hooks`
or
`npm i @busshi/react-hooks`
